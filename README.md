# java-cli-gradle-h2-simple

## Description
A java gradle build, that connects to h2
database.

H2 is a database written in `java`
and is best known for a `dev` only
use.

## Tech stack
- java
- gradle
- h2 driver

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
